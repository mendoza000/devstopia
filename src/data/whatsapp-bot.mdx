La primera vez que nacio esta necesidad de tener que automatizar los mensajes de Whatsapp fue cuando empece a promocionar unos productos mediante este servicio de mensajeria, tenia que enviar publicidad a varios grupos varias veces al dia, lo cual se volvio un tema fastidioso para mi... Fue cuando tome la iniciativa de investigar sobre alguna forma de hacer esto de forma automatica y con un lenguaje que dominara lo suficiente para no frutrarme, ya que fue cuando llevaba poco tiempo programando, encontre la libreria **Whatsapp-web.js** y soluciono mi problema. Pude crear un CLI junto a **Enquierer**.
